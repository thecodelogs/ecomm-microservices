Kafka Topics Map
// event-driven flows via Kafka topics

order.placed
→ payment-service (charge)
→ notification-service (confirm email)

payment.completed
→ order-service (mark paid)
→ shipping-service (fulfil)

payment.failed
→ order-service (cancel)
→ notification-service (alert user)

order.shipped
→ notification-service (tracking)

user.registered
→ notification-service (welcome)
→ analytics-service (track)
